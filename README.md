# 수직과외 GitHub Pages 정적 사이트

GitHub Pages 또는 일반 HTML 호스팅에 바로 업로드할 수 있는 루트 기준 패키지입니다.

## 포함 파일

- `index.html`
- `about/` — 수직과외 소개

- `regions/` — 특별시와 광역시 하위 구·군 지역 페이지
- `assets/css/style.css`
- `assets/js/main.js`

## GitHub Pages 업로드

1. ZIP 파일을 압축 해제합니다.
2. 내부 파일과 폴더 전체를 GitHub 저장소 루트에 업로드합니다.
3. `Settings > Pages`에서 배포 브랜치와 루트 경로를 선택합니다.
4. 배포 URL 확정 후 `sitemap.xml`, `robots.txt`, 각 HTML의 canonical URL placeholder를 실제 URL로 교체합니다.
5. 네이버 서치어드바이저에서 사이트 등록, 소유 확인, 사이트맵 제출, 수집 요청 순서로 진행합니다.
